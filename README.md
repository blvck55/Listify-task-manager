Listify is a modern Task Management Android application built using Kotlin and Jetpack Compose.  
It allows users to manage tasks efficiently with role-based access (User / Admin).

---

##  Features

### Authentication
- User Login
- User Registration
- Admin Login
- Role-based access control

### User Dashboard
- View tasks
- Add new tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Task history view
- Role badge display (USER / ADMIN)

###  Admin Dashboard
- View all tasks
- Mark tasks complete
- Delete tasks
- View total tasks count
- View completed tasks count
- System usage report page

###  UI & UX
- Jetpack Compose UI
- Dark Mode / Light Mode / System Theme
- Background image design
- Glass effect cards
- Navigation animations
- Micro-interaction login animation
- Responsive layout (supports different screen sizes)

---

##  Architecture

- Kotlin
- Jetpack Compose
- Navigation Compose
- Material 3
- In-memory Task Store
- Role-based Authentication Manager
